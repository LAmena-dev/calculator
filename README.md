# Calculator
An [interactive calculator webpage](https://www.theodinproject.com/lessons/foundations-calculator) done through HTML, CSS and JavaScript.

## Technology used
* HTML
* CSS (Flexbox)
* JavaScript (Functions, DOM Manipulation)

## Features
* Basic arithmetic
* Decimal points
* Calculator screen clear
* Delete button
